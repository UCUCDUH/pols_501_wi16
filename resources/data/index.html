    List of Data

    Here are links to some datasets potentially of interest to political scientists. However, the data collection process in academia is disaggregated, with many data sets collected by researchers for specific purposes and distributed by the researcher with their paper. See also the Washington Library Data Research Guide

    Repositories

    • Harvard Dataverse Contains replication data for many political science articles (among other data). The R package dvn allows for direct download and upload of data to and from the Dataverse.

    • ICPSR Collection of many political science and economics datasets.

    • Quandl A collection of financial and economic data. The R package Quandl allows for easy access to Quandl data within R.

    • General Social Survey(GSS)

    Economics

    • World Development Indicators (WDI). The R package WDI allows for downloading data.
    • Penn World Table

    American Politics

    • FRED Collection of U.S. economic data by the Federal Reserve Bank of St. Louis.
    • data.gov U.S. government open data
    • U.S. Census Bureau
    • NOMINATE Ideal Points of U.S. Congress

    Comparative Politics

    • Policy Agendas Project http://www.policyagendas.org/page/datasets-codebooks
    • Varieties of Democracy (VDEM) project http://kellogg.nd.edu/projects/vdem/
    • Polity Project data Political regime characteristics
    • Database of Political Institutions

    International Relations

    • Paul Hensel maintains a list of Political Science Datasets http://www.paulhensel.org/data.html
    • UCDP/PRIO Armed Conflict Datasets on armed conflict
    • ACLED (Armed Conflict Location and Event Data Project) Real-time data on political violence in African and Asian states
    • OpenEvent Data/Phoenix Data Project
    • Data version of the CIA World Factbook: http://jmatchparser.sourceforge.net/factbook/
    • Paul Hensel’s List of International Relations Data
    • Issue Correlates of War (ICOW)

    Open Data / Open Government Datasets

    • Sunlight Foundation Provides open data on the U.S. government. Data on Congress, state legislatures, campaign finance, contributions, dockets.
    • ROpenGov has R packages to access various government related data. For packages to access data on the web from within R see the Web Technologies and Services CRAN Task View.
    • Awesome Public Datasets by caesar0301
    • Vox stories on github
    • NY Times Upshot on github
    • Guardian Data Blog here
    • Fivethirtyeight data on github
    • Catalog of open government data around the world at DataPortals.org
    • Sunlight Foundation Datasets focusing on US government transparency
    • ProPublica
    • NYTimes Campaign Finance API
    • UCI Machine Learning Repository : repository of machine learning datasets
    • Open Data on Github
